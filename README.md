# ayuhealth-unified-compliance-framework
Unified Compliance Framework for AyuHealth Tech Solutions - Rationalized ISO 27001, SOC 2, HIPAA, SOX, and NIST CSF 2.0 into ~315 common controls using Adobe CCF. Demonstrates "Test Once, Comply Many" philosophy with risk assessment, IAM hardening (FIDO2/YubiKeys), immutable AWS backups, and audit-ready documentation.
