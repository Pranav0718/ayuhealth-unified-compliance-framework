# INFO 7374 Unified Compliance Framework Presentation

## Overview
This is the official presentation for the final project "Designing a Unified Compliance Framework for AyuHealth Tech Solutions" presented in December 2025.

## Presentation Contents

### Key Sections:
1. **Organization and Objective** - AyuHealth Profile and IPO readiness goal
2. **Regulatory Landscape** - ISO 27001, SOC 2, HIPAA, SOX, NIST CSF
3. **Methodology** - CCF approach to rationalize 4,300 requirements into ~315 common controls
4. **Compliance Mapping** - Cross-framework control alignment
5. **Risk Assessment** - Threat landscape and gap analysis
6. **Proposed Controls** - IAM, Data Protection, Operations, Risk Management
7. **12-Month Roadmap** - Phased implementation plan
8. **Executive Dashboards** - Governance and SOC dashboards

## Key Takeaways

- **Test Once, Comply Many**: A unified framework approach that eliminates duplicate testing and audit fatigue
